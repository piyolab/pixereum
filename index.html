What's this?

This is an experimental Ethereum based smart contract project.

  • There's 10000 pixels (x=100, y=100) on Pixereum.
  • Each pixel is managed on the Ethereum's blockchain by a smart contract.
  • Inspired by http://www.milliondollarhomepage.com

What you can do?

  • You can be a holder of pixels by sending Ether to the smart contract.
  • Holders of each pixel can set any color they want.
  • Holders can also set any message to each pixel they own.
  • Holders can set any price to each pixel they own and sell it.

How to interact with pixels?

  • Try clicking each pixel.
  • Using MetaMask is recommended.
