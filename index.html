What's this?

This is an experimental Ethereum based smart contract project to learn smart contract.

Inspired by http://www.milliondollarhomepage.com

What you can do?

  • You can be a holder of pixels by sending Ether to the smart contract.
  • Holders of each pixels can set any color they want.
  • They can also set a message to each pixels they own.
  • Yu can set any price to a pixel you own and sell it.
